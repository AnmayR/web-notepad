# Web Notepad

A **web-based Notepad clone** built with HTML, CSS, and JavaScript that works just like **Windows Notepad**.  
It includes menus, keyboard shortcuts, word wrap, font settings, and a status bar â€“ making it a perfect tool for learning HTML and practicing coding in a familiar environment.
